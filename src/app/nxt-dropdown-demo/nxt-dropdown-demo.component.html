üìñ NXT Dropdown Component Guide

The NXT Dropdown component is a powerful, customizable dropdown/select component built for Angular applications. It provides advanced features like multiple selection, confirmation mode, search functionality, and excellent performance with large datasets.

üöÄ Installation & Setup

1. Import the Module

import {{ '{' }} NxtDropdownModule {{ '}' }} from './nxt-dropdown/nxt-dropdown.module';

  @NgModule({{ '{' }}
  imports: [
    NxtDropdownModule,
    // ... other imports
  ]
{{ '}' }})
export class AppModule {{ '{' }} {{ '}' }}

2. Basic Usage

<nxt-dropdown
  [(ngModel)]="selectedValue"
  [options]="options"
  placeholder="Select an option">
</nxt-dropdown>

‚öôÔ∏è Input Properties

Property Type Default Description
options NxtDropdownOption[] [] Array of options to display in the dropdown
placeholder string 'Select an option' Placeholder text when no option is selected
multiple boolean false Enable multiple selection mode
confirmation boolean false Show Apply/Cancel buttons for multiple selection
required boolean false Mark the field as required for validation
disabled boolean false Disable the select component
searchable boolean false Enable search functionality within options
searchPlaceholder string 'Search options...' Placeholder text for search input
minSearchLength number 0 Minimum characters required to trigger search
panelClass string '' Additional CSS class for dropdown panel
config NxtDropdownConfig {{ '{' }}{{ '}' }} Configuration object (alternative to individual properties)
strictConfigMode boolean false When true, prevents mixing direct properties with config object

üì° Output Events

Event Type Description
selectionChange EventEmitter<any> Emitted when selection changes (single value or array for multiple)
optionSelect EventEmitter<NxtDropdownOption> Emitted when an individual option is selected
optionDeselect EventEmitter<NxtDropdownOption> Emitted when an individual option is deselected
dropdownOpen EventEmitter<void> Emitted when dropdown opens
dropdownClose EventEmitter<void> Emitted when dropdown closes

üìä Data Structure

NxtDropdownOption Interface

interface NxtDropdownOption {{ '{' }}
  value: any;           // The value of the option
  label: string;        // Display text for the option
  disabled?: boolean;   // Whether the option is disabled
  group?: string;       // Group label for grouped options
  icon?: string;        // Icon class or URL
  description?: string; // Additional description text
{{ '}' }}

NxtDropdownConfig Interface

interface NxtDropdownConfig {{ '{' }}
  options?: NxtDropdownOption[];    // Array of options
  placeholder?: string;             // Placeholder text
  disabled?: boolean;               // Disable the component
  required?: boolean;               // Mark as required
  multiple?: boolean;               // Enable multiple selection
  confirmation?: boolean;           // Show confirmation buttons
  panelClass?: string;              // Additional CSS class
  searchable?: boolean;             // Enable search functionality
  searchPlaceholder?: string;       // Search input placeholder
  minSearchLength?: number;         // Minimum search length
{{ '}' }}

üîí Configuration Validation

Strict Configuration Mode

The component supports a strict configuration mode that prevents mixing direct input properties with configuration objects. This helps maintain consistency and prevents configuration conflicts.

‚úÖ Valid Usage (Strict Mode)

// Config object only
<nxt-dropdown
  [strictConfigMode]="true"
  [config]="myConfig"
  [(ngModel)]="value">
</nxt-dropdown>

// Direct properties only
<nxt-dropdown
  [strictConfigMode]="false"
  [options]="myOptions"
  [placeholder]="'Select option'"
  [multiple]="true"
  [(ngModel)]="value">
</nxt-dropdown>

‚ùå Invalid Usage (Will Show Console Error)

// Mixing config object with direct properties
<nxt-dropdown
  [strictConfigMode]="true"
  [config]="myConfig"
  [placeholder]="'This will cause an error'"
  [multiple]="true"
  [(ngModel)]="value">
</nxt-dropdown>

Console Error Example
